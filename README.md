Este projeto foi desenvolvido para a disciplina de Interface Humano-Computador com o objetivo de aplicar a 1ª Heurística de Nielsen, que trata da Visibilidade do Status do Sistema. O software simula uma varredura de hardware em C# utilizando o comando Thread.Sleep para criar intervalos de tempo realistas e o caractere especial \r para atualizar as mensagens na mesma linha do terminal. Essa abordagem garante que o usuário receba feedback constante sobre o que o computador está processando, evitando a sensação de que o programa travou ou parou de responder. O projeto demonstra como pequenos detalhes técnicos na interface de linha de comando podem melhorar significativamente a experiência do usuário e a percepção de controle sobre o sistema.
